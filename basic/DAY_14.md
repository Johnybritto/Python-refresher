## Tiny Project Idea
Build a small student summary program.

The program should use:
- one string for a student name
- one list for subjects
- one dictionary for student details
- one loop to print the subjects
- one function to print a greeting or summary

## Example Thinking
You can build it in tiny steps:

1. Create a string like `"Ravi"`
2. Create a list like `["Math", "Science", "English"]`
3. Create a dictionary like `{"name": "Ravi", "age": 12}`
4. Write a function that prints a short greeting
5. Use a loop to print each subject

## Hint
Do not try to make it fancy.

A small and correct program is better than a big confusing one.
