# 14-Day Python Learning Plan

## Goal
Build a strong beginner foundation in Python in 14 days with 1 hour of study per day.

## Study Rules
- Learn in tiny steps.
- Use hints before full solutions.
- Do one exercise at a time.
- Review each code attempt for correctness, readability, time complexity, and space complexity.
- End each session by noting what to study next.

## Daily Structure (1 Hour)
- 10 min: Read the day's notes
- 5 min: Quick revision of one older core pattern like string reversal or time/space complexity
- 15 min: See examples and ask questions
- 25 min: Solve one small exercise
- 10 min: Review mistakes and log progress

## Roadmap

### Day 1: Assessment + Python Basics
- Check current level with 4 short questions
- Learn variables, `print()`, comments, and basic input/output
- Practice reading simple code

### Day 2: Data Types and Expressions
- Integers, floats, strings, booleans
- Basic operators: `+`, `-`, `*`, `/`, `//`, `%`
- Predict expression results

### Day 3: Strings I
- Indexing
- Slicing
- Common methods like `.lower()`, `.upper()`, `.strip()`
- Keep revising string reversal by reading indexes from right to left

### Day 4: Strings II
- Building strings with f-strings
- Counting, finding, replacing
- Small text-cleaning tasks
- Revisit string reversal with slicing and loop thinking

### Day 5: Lists I
- Creating lists
- Indexing and slicing
- `append()`, `pop()`, `len()`

### Day 6: Lists II
- Looping through lists
- Updating values
- Intro to list problem solving

### Day 7: Dictionaries I
- Key-value pairs
- Accessing, adding, updating values
- When to use a dict instead of a list

### Day 8: Dictionaries II + Sets
- Dictionary methods
- Counting with dictionaries
- Set basics: uniqueness, membership tests

### Day 9: Conditionals
- `if`, `elif`, `else`
- Comparison operators
- Logical operators: `and`, `or`, `not`

### Day 10: Loops
- `for` loops
- `while` loops
- Loop patterns for counting and searching
- Reverse a string with a backward loop using `range(..., ..., -1)`
- Keep revising reverse string with a `for` loop

### Day 11: Functions I
- Defining functions
- Parameters and return values
- Breaking code into small reusable parts

### Day 12: Functions II + Debugging
- Scope basics
- Reading tracebacks
- Using `print()` for debugging

### Day 13: Complexity Basics
- Time complexity intuition
- Space complexity intuition
- Compare simple list and dict operations
- Keep revising time vs space complexity in plain words

### Day 14: Mini Review Project
- Combine strings, lists, dicts, loops, and functions
- Solve one practical mini problem
- Review strengths, weak spots, and next steps

## What Success Looks Like
By the end of Day 14, you should be able to:
- Read short Python programs confidently
- Write small programs with strings, lists, dicts, and sets
- Use functions to organize code
- Debug common beginner mistakes
- Explain simple time and space complexity ideas
- Reverse a string using both slicing and a loop
