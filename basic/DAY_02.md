## Tiny Lesson

### 1. Integers
Integers are whole numbers.

```python
age = 21
score = 100
```

### 2. Floats
Floats are decimal numbers.

```python
price = 19.5
temperature = 36.7
```

### 3. Strings
Strings are text values.

```python
name = "Ravi"
city = "Mumbai"
```

### 4. Booleans
Booleans are only `True` or `False`.

```python
is_logged_in = True
is_raining = False
```

## Operators

```python
print(7 + 3)   # 10
print(7 - 3)   # 4
print(7 * 3)   # 21
print(7 / 2)   # 3.5
print(7 // 2)  # 3
print(7 % 2)   # 1
```

### Important Idea
- `/` gives division result as a float
- `//` gives floor division
- `%` gives the remainder
