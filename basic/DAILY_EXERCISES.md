# Daily Exercises

Use exactly one small exercise per day.

## Recurring Revision
Keep revisiting this pattern every few days:
- reverse a string with a loop
- reverse a string with a `for` loop
- reverse a string with slicing
- explain time complexity in simple words
- explain space complexity in simple words

## Day 1
Write a program that:
- stores your name in a variable
- stores your favorite number in another variable
- prints: `My name is <name> and my favorite number is <number>`

## Day 2
Write a program that:
- stores two numbers in variables
- prints their sum
- prints their product
- prints the remainder when the first number is divided by the second

## Day 3
Given a word, print:
- the first character
- the last character
- the first three characters

## Day 4
Given a sentence, print:
- the sentence in uppercase
- the sentence in lowercase
- the sentence with extra spaces removed

## Day 5
Create a list of 4 fruits and:
- print the first fruit
- print the last fruit
- add one more fruit

## Day 6
Create a list of numbers and:
- print each number
- print the total count using `len()`
- change one value in the list

## Day 7
Create a dictionary for one student with:
- `name`
- `age`
- `grade`

Then print each value.

## Day 8
Create a list with repeated numbers and:
- convert it to a set
- print the unique values

## Day 9
Write a program that:
- stores a number
- prints `"positive"` if it is greater than 0
- otherwise prints `"zero or negative"`

## Day 10
Use a loop to print numbers from `1` to `5`.

## Day 11
Write a function that takes a name and prints a greeting.

## Day 12
Write a small buggy program, run it, and note:
- the error message
- what caused it
- how you fixed it

## Day 13
Compare these operations in simple words:
- finding an item in a list
- checking whether a key exists in a dictionary

## Day 14
Build a mini program that uses:
- one string
- one list
- one dictionary
- one loop
- one function
